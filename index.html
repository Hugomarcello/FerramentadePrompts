🧩 O que é uma Skill no Claude Code?
Uma Skill é um arquivo SKILL.md dentro de uma pasta específica. Quando você abre o Claude Code no terminal, ele lê essa pasta e adiciona a skill ao seu conjunto de ferramentas automáticas. Você pode invocar uma skill digitando /nome-da-skill ou deixar o Claude usá-la automaticamente quando detectar que é necessário.
🔁
Reutilizável
Crie uma vez e use em todos os seus projetos, sem copiar e colar prompts repetidamente
📂
Por arquivo
É só um arquivo de texto com instruções. Nada de código complexo ou configurações difíceis
⚡
Automática
O Claude decide quando usar com base na descrição, ou você chama com /nome
🌍
Open standard
Segue o padrão AgentSkills.io — funciona também em outras ferramentas de IA
🗂️ Onde ficam as skills?
Pessoal: ~/.claude/skills/nome-da-skill/SKILL.md → disponível em todos os seus projetos
Por projeto: .claude/skills/nome-da-skill/SKILL.md → só neste projeto (compartilhável via git)
Empresa: gerenciado pelo administrador → todos os usuários da organização
💡 Quando criar uma Skill?
Quando você perceber que está copiando e colando o mesmo prompt mais de 2 vezes. Se virou rotina, vira Skill.
📄 Etapa 2 — Estrutura do SKILL.md
Todo arquivo SKILL.md tem duas partes obrigatórias separadas por ---. Entender essa estrutura é o único requisito técnico para criar uma skill.
📄 Estrutura do SKILL.md Gabarito
---# ← início do frontmatter (obrigatório)
name: nome-da-skill# nome exibido nas listagens (opcional)
description: O que faz e quando usar. Ex: Use quando...# ⚠️ CRÍTICO — é o que o Claude lê para decidir se usa
disable-model-invocation: false# true = só você invoca com /nome (não automático)
---# ← fim do frontmatter
## Instruções# ← aqui vão as instruções em markdown
Faça X, depois Y, depois Z...
!`comando-do-terminal`# ← injeta saída de comando em tempo real (opcional)
🔑 O campo description é o mais importante.
É ele que o Claude lê para decidir se vai usar a skill automaticamente. Seja específico: coloque casos de uso, gatilhos, e exemplos de quando usar. Ex: "Use quando o usuário perguntar sobre mudanças no git, quiser um commit message, ou pedir para revisar o diff."
⚡ Injeção dinâmica de contexto (!`comando`)
Antes de o Claude ver o conteúdo da skill, o Claude Code executa o comando e substitui a linha pelo resultado. Assim a skill já chega com dados reais — não chutes. Ex: !`git diff HEAD`, !`cat package.json`, !`ls src/`

🚀 Etapa 4 — Instalar e Usar a Skill
Sua skill está pronta! Agora veja como instalá-la e testá-la no terminal.
1
Crie a pasta da skill
mkdir -p ~/.claude/skills/minha-skill
Substitua minha-skill pelo nome que você definiu. Para skill de projeto: use .claude/skills/ na raiz do projeto.
2
Crie o arquivo SKILL.md
touch ~/.claude/skills/minha-skill/SKILL.md
Abra o arquivo em qualquer editor de texto (VS Code, Notepad, nano) e cole o conteúdo gerado no preview abaixo.
3
Abra o Claude Code
claude
Se o Claude Code já está aberto, não precisa reiniciar — ele detecta mudanças nas skills automaticamente em tempo real.
4
Teste a skill
/minha-skill
Ou deixe o Claude invocar automaticamente fazendo uma pergunta que case com a descrição da sua skill.
5
Liste todas as suas skills
claude --list-skills
Ou dentro de uma sessão Claude Code, use /skills para ver todas as skills carregadas.

🌐 Skills no Claude.ai (sem terminal)
No Claude.ai, não existe o conceito de "Skill" como arquivo. O equivalente é usar as Instruções do Projeto — um texto fixo que o Claude sempre lê antes de cada conversa dentro daquele projeto. Use a aba 🗂️ Projeto Claude desta ferramenta para montar esse prompt.
📋
Aba Gabarito
Para prompts pontuais e únicos, sem contexto fixo
🗂️
Aba Projeto Claude
Para contexto persistente que a IA sempre deve lembrar — o equivalente a uma Skill no Claude.ai
⚡
Aba Etapas
Para fluxos condicionais com portas lógicas, onde a IA segue passos definidos
💻
Claude Code
Instale o Claude Code no terminal para usar Skills reais como arquivos SKILL.md reutilizáveis
🔗 Como instalar o Claude Code:
1. Tenha Node.js instalado no seu computador
2. No terminal: npm install -g @anthropic-ai/claude-code
3. Autentique: claude (abre o navegador para login)
4. Pronto — agora você pode criar Skills de verdade! Volte para a opção Claude Code acima.
